Suggested order

First PRs

  1. FoundationFonts, tokens.css, Tailwind config, <html dir="rtl"> shell. Empty Home renders the wordmark.
  2. PrimitivesButton, Pill, Term, CheckTile, Input. Match design system 1:1. Test in Storybook.
  3. Stage shellTopBar + ProgressStrip + StageHeader. Hard-code stage 4 data. Visual parity check.
  4. Education layerBriefing, Expand, Questions, BottomSheet.
  5. Media layerPhoto, RefGallery, VideoCard.
  6. Active bake store + persistenceZustand or Context store, persist to localStorage on every mutation.
  7. Remaining 11 stagesData-driven once template is solid.
  8. Recipe form + presets · Completion · Cheat sheetMostly form work + read-only views.
Reminder

The non-negotiables

Hebrew only. RTL throughout. <html dir="rtl">.
Logical properties only: ms-/me-/ps-/pe-. Never ml-/mr-/left-/right-.
Touch targets ≥ 44×44. Always.
Numbers stay LTR inside Hebrew: <span dir="ltr">75%</span>
No animation libraries. CSS + plain JS. Respect prefers-reduced-motion.
